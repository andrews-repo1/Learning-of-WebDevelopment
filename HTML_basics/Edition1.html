The Goal of HTML

HTML is a markup language whcih annotates the text to define how it is displayed by the web browser. It contains two basic concepts: elements and tags.

The limitations

HTML could define the page structure and the text contents, but it can't freely modify the visual appearance and create interactive links, which could be accomplished by combining with CSS and JavaScript.
We could use CSS to improve the visual appearance of our webpage and JavaScript to provide interactive features.
